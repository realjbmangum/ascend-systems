## Typography

### Strategy

Modern tech brand with authority. Sans-serif headings for clean tech energy. A secondary monospace font for data/stats creates a "systems" feel that reinforces the name.

### Heading Font: Inter

| Property | Value |
|----------|-------|
| **Family** | `'Inter', system-ui, sans-serif` |
| **Weights** | 600 (SemiBold), 700 (Bold), 800 (ExtraBold) |
| **Why** | Clean, modern, designed for screens. Used by Vercel, Linear, GitHub. Feels 2026. |

### Body Font: Inter

| Property | Value |
|----------|-------|
| **Family** | `'Inter', system-ui, sans-serif` |
| **Weights** | 400 (Regular), 500 (Medium) |
| **Why** | Single font family keeps it tight and systematic. Weight contrast creates hierarchy. |

### Monospace Accent: JetBrains Mono

| Property | Value |
|----------|-------|
| **Family** | `'JetBrains Mono', monospace` |
| **Weights** | 400 (Regular), 500 (Medium) |
| **Why** | Used for stats, data points, technical details. "2,847 calls handled" in monospace feels like a live dashboard. Reinforces the "systems" brand. |

## Logo Direction

### Concept: Geometric Wordmark + Arrow Mark

**The wordmark:** "ASCEND SYSTEMS" set in Inter ExtraBold (800), all caps, tight letter-spacing (-0.03em). Clean, modern, confident.

**The mark:** A simple upward-pointing geometric element — could be an abstract arrow, chevron, or ascending line graph. Built from the same geometric language as the typography. Works as a standalone favicon/icon.

**Color applications:**

| Context | Logo Color | Background |
|---------|-----------|------------|
| **Primary (dark bg)** | White wordmark + orange mark | Charcoal `#1C1C1E` |
| **Primary (light bg)** | Charcoal wordmark + orange mark | Off-White `#FAFAF8` |
| **Single color (dark bg)** | White | Charcoal |
| **Single color (light bg)** | Charcoal | White |
| **Favicon** | Orange mark on charcoal | Dark background |

### Logo Sizing

| Context | Minimum Width |
|---------|--------------|
| **Full wordmark (web)** | 160px |
| **Full wordmark (print)** | 1.75" |
| **Icon/mark only (web)** | 24px |
| **Icon/mark only (print)** | 0.375" |
| **Clear space** | Equal to the height of the "A" in ASCEND on all sides |

### What the Logo Should NOT Be
- No lighthouses, mountains, or literal "ascending" imagery
- No gradients or 3D effects
- No circuit boards, robots, or AI cliches
- Nothing that requires explanation
- Nothing with more than 2 colors

## Brand Voice Quick Reference

### How Ascend Systems Talks

| Context | Tone | Example |
|---------|------|---------|
| **Website hero** | Confident, minimal | "Every call, handled." |
| **Website body** | Clear, specific | "When you're on a job and a customer calls, Ascend picks up, gets their info, and books the appointment. You get a text with the details." |
| **Chamber intro** | Personal, direct | "I'm Brian with Ascend Systems. We answer phones for local businesses using AI so they never miss a customer." |
| **Email outreach** | Warm, local | "Hey Dave — I run Ascend Systems here in Mint Hill. I help businesses stop missing calls. Want to see how it works?" |
| **Sales follow-up** | Helpful, low-pressure | "Just checking in. Your AI handled 47 calls last week and booked 12 appointments. Here's the breakdown." |

### Words We Use
- Handled, covered, managed, built, running, automated
- Calls, leads, appointments, customers, revenue
- Set up, configured, monitored, optimized

### Words We Don't Use
- Leverage, synergy, ecosystem, paradigm, disrupt
- AI-powered (in headlines — it's implied)
- Never miss a call (6+ competitors use this exact phrase)
- Revolutionary, cutting-edge, game-changing
